# 楽譜がっちゃんこ

画像やPDFの楽譜をブラウザ上で並べ替え、横につないで保存できるWebアプリです。

## 公開サイト

[楽譜がっちゃんこを使う](https://moerita-ai.github.io/score-gatchanko/)

## 主な機能

- 画像・PDFの読み込み
- ページや楽譜のドラッグ＆ドロップ並べ替え
- つないだ楽譜のプレビュー
- PDF・画像としての保存
- スマートフォン表示への対応

ファイル処理はブラウザ内で行われ、読み込んだ楽譜はサーバーへ送信されません。

## 開発

Node.js 22.13.0以上とpnpmを使用します。

```bash
pnpm install
pnpm dev
```

本番用ビルド:

```bash
pnpm build
```
